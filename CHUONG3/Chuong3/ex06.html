CSS CĂN BẢN

CSS là gì?

CSS (Cascading Style Sheets) là ngôn ngữ quy định cách trình bày cho tài liệu HTML, XHTML...

Học CSS cần những gì?

  • Kiến thức cơ bản về HTML
  • Trình soạn thảo CSS
  • Trình duyệt
  • Thời gian thực hành

Những lệnh thường gặp trong CSS

  • Background: background-color, background-image...
  • Font: font-family, font-style, font-weight...
  • Color
